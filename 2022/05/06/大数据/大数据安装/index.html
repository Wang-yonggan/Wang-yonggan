大数据安装

发表于 2022-05-06 更新于 2022-05-06 Changyan：
本文字数： 4k 阅读时长 ≈ 4 分钟

一、Hadoop

1.1 前提

参考 linux怎么安装hadoop-linux运维-PHP中文网

1.2 安装

  1. wget http://mirrors.hust.edu.cn/apache/hadoop/core/stable/hadoop-2.7.1.tar.gz

  2. tar -zxvf hadoop-2.7.1.tar.gz

  3. vim etc/hadoop/core-site.xml

  4. vim mapred-site.xml.template

  5. vim hdfs-site.xml

  6. vim start-dfs.sh 和 stop-dfs.sh

  7. vim start-yarn.sh 和 stop-yarn.sh文件

  8. vim etc/hadoop/hadoop.env.sh

  9. vim /etc/profile 和 hadoop-env.sh

  10. bin/hdfs namenode -format

1.3 验证

  1. yarn http://c2:8088/cluster
  2. hdfs http://c2:50070 hadoop3 改为9870
  3. spark 4040/jobs

1.4 更换版本

目前方案就是重新下载，然后重新配置

二、Hbase

2.1 前提

  1. java
  2. hadoop 并启动

2.2 安装

  1. 找到与Hadoop对应的版本 2.4.x

    https://hbase.apache.org/book.html#hadoop

  2. 找到镜像，解压

  3. vim hbase-env.sh

  4. vim hbase-site.xm

  5. vim /etc/profile

  6. vim regionservers

2.3 启动

  1. ./bin/start-hbase.sh
  2. jps查看
  3. ./bin/hbase shell
  4. list
