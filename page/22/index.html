  1. 每一分钟执行一次 /bin/ls：

  2. 在 12 月内, 每天的早上 6 点到 12 点，每隔 3 个小时 0 分钟执行一次 /usr/bin/backup：

  3. 周一到周五每天下午 5:00 寄一封信给 alex@domain.name：

  4. 每月每天的午夜 0 点 20 分, 2 点 20 分, 4 点 20 分….执行 echo “haha”：

  5. 0 */2 * * * /sbin/service httpd restart  意思是每两个小时重启一次apache 

50 7 * * * /sbin/service sshd start  意思是每天7：50开启ssh服务 

50 22 * * * /sbin/service sshd stop  意思是每天22：50关闭ssh服务 

0 0 1,15 * * fsck /home  每月1号和15号检查/home 磁盘 

1 * * * * /home/bruce/backup  每小时的第一分执行 /home/bruce/backup这个文件 

00 03 * * 1-5 find /home "*.xxx" -mtime +4 -exec rm {} \;  每周一至周五3点钟，在目录/home中，查找文件名为*.xxx的文件，并删除4天前的文件。

30 6 */10 * * ls  意思是每月的1、11、21、31日是的6：30执行一次ls命令

一、简单操作

1.1 文件

  1. ls -a 连同带.的隐藏文件文件一起查看

  2. 解压 tar xzf ..gz

    文件类型 压缩 解压 打包
    tar文件 tar文件不是压缩文件 tar -xvf name.tar tar -cvf name.jar name
    gz文件 tar -zcvf name.tar.gz name tar -zxvf name.gz
    zip文件 zip name.zip name unzip name.zip
    rar文件 rar a name.rar name rar x name.rar 
1. 说明

    1. c建立新的压缩文件
    2. v 显示操作过程
    3. f 指定文件
    4. x 提取文件
