2.2.1 format_string

控制颜色字符、风格字符、标记字

  • 颜色：’b’ 蓝色
  • 字符风格：默认’-‘折线段

2.3 plt.axis

  1. plt.axis([1, 2, 3, 4])

    设置x范围 1-2 y范围3-4

  2. plt.axis(option=‘equal’)

    1. equal：x-y坐标等长
    2. off： 关闭
    3. auto：自动
    4. square：正方形

2.4 plt.explain 说明文字参数

  1. color : 颜色
  2. size：大小
  3. 其他：字体系列，对其方式，行距，粗细，样式，字体名

2.5 plt.subplots 子图

返回一个包含figure和axes对象的元组

figure: 拿到整张图

ax： np.array[axe 对象]

subplots 和subplot 区别

  • subplots 一次性创建并返回所有的子图和其 axe 对象

  • subplot分开子图并拿到指定位置的子图plt.subplot(2, 1, 2) 分成2行1类并拿到第二个ax对象
