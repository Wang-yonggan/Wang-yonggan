四大函数式接口

  1. Consumer

    ​ 消费型：有参无返回

    ​ void accept(T t)

  2. Suppiier 它是一个可以获取任意值的接口而已

    ​ 供给型：无参有返回

    ​ T get()

  3. Functional<T,R>

    ​ 函数型：处理参数

    ​ R apply(T t)

  4. Prediate

    ​ 断定型：有参判定返回bool

    ​ boolean test(T t)
