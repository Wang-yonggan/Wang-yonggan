redis命令

发表于 2022-03-10 Changyan：
本文字数： 603 阅读时长 ≈ 1 分钟

1.基于内存的key-value数据库
2.基于c语言编写的，可以支持多种语言的api //set每秒11万次，取get 81000次
3.支持数据持久化
4.value可以是string，hash， list， set, sorted set

使用场景

  1. 去最新n个数据的操作
  2. 排行榜，取top n个数据 //最佳人气前10条
  3. 精确的设置过期时间
  4. 计数器
  5. 实时系统， 反垃圾系统
  6. pub， sub发布订阅构建实时消息系统
  7. 构建消息队列
  8. 缓存

  1. 切换库 select 1

2.1.2 key操作

注意每个单词要有空格分开

  1. 查看所有 keys * keys nam* 查看所有 nam开头的*
  2. 赋值 set k v
  3. 多赋值 mset k1 k2 v1 v2
  4. 获取值 get k
  5. 多取值 mget k1 k2
  6. 检查是否存在 exists key
  7. key类型 type key
  8. 删除key del key
  9. 删除所有key flushdb
  10. 设置过期时间秒 expire key 10
  11. persist key 删除过期时间
  12. ttl key 查看多少秒过期 -2 已经过期 -1 永不过期
