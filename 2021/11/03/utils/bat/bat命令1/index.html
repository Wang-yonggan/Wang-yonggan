bat常见命令

  1. REM 和 :: 注释

  2. ECHO 和 @

    1. @字符放在命令前将关闭该命令回显，无论此时echo是否为打开状态。

    2. @ echo off 关闭off 并将这句话不会回显

    3. 打印空行 echo.

    4. echo | 管道传参(将echo的输出当作|后面的输入)

    5. 新建文件 echo aaa > a.txt

    6. @echo off 关闭cmd的文件路径

  3. pause

    Echo 其他提示语 & pause > nul

  4. errorlevel

    显示返回码echo %errorlevel%

    正常返回0 错误1

  5. title 设置cmd窗口标题

  6. color 设置cmd颜色

    cmd

    0 = 黑色 8 = 灰色
    1 = 蓝色 9 = 淡蓝色
    2 = 绿色 A = 淡绿色
    3 = 湖蓝色 B = 淡浅绿色
    4 = 红色 C = 淡红色
    5 = 紫色 D = 淡紫色
    6 = 黄色 E = 淡黄色
    7 = 白色 F = 亮白色

  7. mode 配置系统设备

  8. GOTO 和 :

  9. FIND

    /V 显示所有未包含指定字符串的行。
    /C 仅显示包含字符串的行数。
    /N 显示行号。
    /I 搜索字符串时忽略大小写。

    Find常和type命令结合使用

  10. START

    批处理中调用外部程序的命令（该外部程序在新窗口中运行，批处理程序继续往下执行，不理会外部程序的运行状况），如果直接运行外部程序则必须等外部程序完成后才继续执行剩下的指令

    例：start explorer d:\

    调用图形界面打开D盘

  11. assoc 和 ftype

  13. pushd 和 popd

  14. CALL

  15. shift

  16. IF

  17. setlocal 与 变量延迟

  18. ATTRIB 显示或更改文件属性
