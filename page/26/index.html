排序算法

发表于 2022-03-08 更新于 2022-01-06 分类于 算法 Changyan：
本文字数： 538 阅读时长 ≈ 1 分钟

快排

思想: 分治

原理

哨兵机制+分治。

  1. 取最左边为哨兵
  2. 从最右边<–>左边，交替遍历，
  3. 最终将nums分成小于哨兵的部分和大于哨兵的部分

易错

  1. 对于nums,x,y型 void quickSort(int[]nums,int begin,int end)

    每次取的哨兵是nums[begin]而不是nums[0]

合并排序

思想: 分治

原理

分治+merge

易错

  1. 合并是在左右两个数组都排好序后在进行{11.两个有序链表合并问题}

  2. 求mid: (begin + end) / 2;

  3. sort:

    if(begin<end)

    • sort(begin,mid);
    • sort(mid+1,end);
    • merge(begin,mid,end);

  4. merge:

    【begin,mid】 【mid+1,end】两个有序数组的Merge

    ​ 创建一个新的temp{}

特点

​ 归并排序是稳定排序，它也是一种十分高效的排序，能利用完全二叉树特性的排序一般性能都不会太差。java中Arrays.sort()采用了一种名为TimSort的排序算法，就是归并排序的优化版本。每次合并操作的平均时间复杂度为O(n)，而完全二叉树的深度为|log2n|。总的平均时间复杂度为O(nlogn)。而且，归并排序的最好，最坏，平均时间复杂度均为O(nlogn)。
